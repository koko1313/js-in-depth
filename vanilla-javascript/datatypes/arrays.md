# Arrays

1. Declare an array

    ```javascript
    var arr = ['a', 'b', 'c'];

    // supports different types
    var arr = ['a', 0, null];
    ```

2. `.push()` and `.pop()`

    ```javascript
    // push element to array
    arr.push(1);
    arr.push('a');

    // pop the last element from the array
    arr.pop();
    ```

3. Get an element by index

    ```javascript
        console.log(arr[0]); // print the first element of the array
    ```
